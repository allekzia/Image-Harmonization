# Image-Harmonization

This repository contains the implementation and analysis of a Spatial-Separated Attention Module (S2AM) for image harmonization, trained on the HCOCO dataset

## Structure
- **Report:** `Image_Harmonization.pdf` (Full academic analysis)
- **Code:** `src/ImageHarmonization.ipynb` (Jupyter Notebook)
- **Data:** [Download HCOCO Dataset (Dropbox)](https://www.dropbox.com/scl/fo/eul5kp3qle6qtkslta7fz/AHR4VsV-5Kz2dLvWpUDnzas?rlkey=r7u9zrh1h5qv4l9hvidbh18xl&e=1&st=7ll8gcdn&dl=0)

## Quick Results
Our model achieved a Test fMSE of **0.1304**, outperforming the U-Net baseline despite training on constrained hardware (Tesla T4)
